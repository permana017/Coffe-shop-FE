# PermanaCoffe

Website Coffeshop for simple Orders.

## About The Project

The Coffeeshop app lets users browse and order food and drink menus, pay online, pre-order, and
leave reviews. It's an efficient solution for cafe-goers who want to skip the queue and enjoy their favorites.
